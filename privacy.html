1. Quién somos

TiroLog es una aplicación web progresiva (PWA) para el registro de sesiones de tiro deportivo, desarrollada por Sapemaib. Puedes contactarnos en info@sapemaib.dev.

2. Qué datos recoge la aplicación

TiroLog recoge únicamente los datos que el propio usuario introduce:

  • Sesiones de tiro: fecha, hora, puntuación, modalidad y notas opcionales.
  • Modalidades de tiro creadas por el usuario.
  • Registros de compras de material relacionado con el tiro.

La aplicación no recoge datos personales de identificación (nombre, dirección, teléfono), datos de ubicación, ni información del dispositivo más allá de lo necesario para el funcionamiento técnico de la PWA.

3. Dónde se almacenan los datos

Todos los datos se almacenan localmente en el dispositivo del usuario mediante IndexedDB. Los datos no se envían a ningún servidor propiedad de Sapemaib.

De forma opcional, el usuario puede activar la copia de seguridad automática en Google Drive. En ese caso, los datos se guardan en la carpeta privada de la app dentro de la cuenta de Google Drive del propio usuario (appDataFolder). Sapemaib no tiene acceso a esos archivos en ningún momento.

4. Uso de Google Drive

Si el usuario activa la integración con Google Drive, TiroLog solicita acceso al scope https://www.googleapis.com/auth/drive.file, que permite crear y actualizar únicamente los ficheros que la propia app ha creado, sin acceder a ningún otro fichero del Drive del usuario.

El token de acceso se almacena temporalmente en el dispositivo y se utiliza exclusivamente para subir y descargar el fichero de copia de seguridad de TiroLog. El uso de los datos obtenidos a través de las APIs de Google cumple con la Política de datos de usuario de los servicios de API de Google.

5. Compartición de datos con terceros

TiroLog no vende, alquila ni comparte ningún dato del usuario con terceros.

6. Cookies y rastreo

TiroLog no utiliza cookies propias ni tecnologías de rastreo. No integra herramientas de analítica ni publicidad.

7. Derechos del usuario

  • Exportar: desde la pestaña "Backup" puede descargar todos sus datos en formato JSON.
  • Eliminar: borrar la aplicación del navegador elimina todos los datos locales.
  • Desconectar Drive: desde la pestaña "Backup" puede revocar el acceso a Google Drive.

8. Cambios en esta política

Si se realizan cambios relevantes, se actualizará la fecha de la cabecera y se notificará mediante la propia aplicación cuando sea posible.

  • Purchase records for shooting-related equipment.

The app does not collect personal identification data (name, address, phone), location data, or device information beyond what is technically required for the PWA to function.

All data is stored locally on the user's device using IndexedDB. Data is never sent to any server owned by Sapemaib.

If the user enables Google Drive integration, TiroLog requests the https://www.googleapis.com/auth/drive.file scope, which allows creating and updating only the files the app itself has created, without accessing any other files in the user's Drive.

The access token is stored temporarily on the device and used exclusively to upload and download TiroLog's backup file. Data use complies with the Google API Services User Data Policy.

5. Data sharing with third parties

TiroLog does not sell, rent, or share any user data with third parties.

6. Cookies and tracking

TiroLog does not use cookies or tracking technologies. It does not integrate analytics or advertising tools.

7. User rights

  • Export: from the "Backup" tab you can download all your data as JSON.
